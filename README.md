# 🛒 E-Commerce Sales & User Behaviour Analysis

Analysis of 99,441 real e-commerce orders using the Olist Brazilian E-Commerce dataset, uncovering sales trends, customer behaviour, anomalies, and seller performance.

## 📊 Key Findings
- **Peak month:** November 2017 (Black Friday) — 7,289 orders, the highest single month
- **Top category:** Bed & Bath Table — 11,115 orders across the platform
- **Average order value:** R$137.04
- **Customer insight:** 55% of orders fall in the mid-value range (R$50–R$200)
- **Top seller revenue:** R$229,472 across 1,156 orders
- **Anomalies detected:** 3 months flagged during platform launch period (Sept–Dec 2016)

## 🔍 Analysis Sections
1. **Data Loading & Cleaning** — 9 datasets, 99,441 orders, datetime parsing, missing value handling
2. **Sales Trend Analysis** — Monthly order volumes across 2016–2018
3. **Product Category Analysis** — Top 10 categories by order volume
4. **Customer Segmentation** — Order value segments (Low / Mid / High)
5. **Anomaly Detection** — Statistical threshold method (1.5 std dev) to flag unusual months
6. **Seller Performance** — Top 10 sellers ranked by revenue and order count
